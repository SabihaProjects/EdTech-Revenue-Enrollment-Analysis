# EdTech-Revenue-Enrollment-Analysis

To showcase skills in business analytics, data modeling, and dashboard reporting using Power BI.  
This project analyzes monthly enrollment & revenue performance across different EdTech course categories.

It helps identify top-growing learning segments, seasonal patterns, and student engagement trends to support data-driven decisions.

###  Tools Used:
- Power BI
- Data Cleaning & Modeling
- Bar, Line & Pie Chart Visualization

###  Files Included:
- Education Platform Dashboard.pbix
- Education Platform.csv
- Documentation (README)
